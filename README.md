
Cybersecurity Awareness Chatbot for South African campaign

Overview.
This is a conversational chatbot that is made to educate South Africans about cybersecurity threats like Phishing, Password Safety, Safe Browsing, Social Engineering and protection from possible malware.

The features included in this project/ChatBot include:
A voices greeting when the project starts up,
and a beautiful ASCII Art logo display with the ChatBot name,
With personalised user interactions.
This chatbot covers cybersecurity education with multiple topics, 
along with a enhancesd console UI with appropriate colors and borders.

Topics Covered
- Password Safety
- Phishing Detection
- Safe Browsing
- Social Engineering
- Malware Protection
- Two-Factor Authentication
